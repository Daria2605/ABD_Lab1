# BigDataSnowflake

Анализ больших данных - лабораторная работа №1
**Тема:** нормализация исходных данных в аналитическую модель типа **звезда / снежинка**

Выполнила: Жгенти Дарья Никитична
Группа: М8О-308Б-23

Алгоритм запуска:
1. Клонировать репозиторий.
2. Поднять postgres с помощью команды `docker compose up -d`.
3. Выполнить source_table.sql скрипт
4. Выполнить DDL.sql скрипт
5. Выполнить DML.sql скрипт
